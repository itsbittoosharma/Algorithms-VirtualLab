Experiment 1

Consider the following set of algorithms, whose worst-case time complexity ranges from O(log(n)) for Binary search to O(2 n ) for Permutations generation:

  • Binary and sequential search
  • Quicksort
  • Matrix addition
  • Matrix multiplication
  • Subset generation
  • Permutations generation

Study how, for gradually increasing computation time limits, the largest input size (n) solvable by each algorithm varies with the time limit provided. Plot a graph of execution time (X-axis) versus problem size (Y-axis) for these algorithms and comment on the results.
